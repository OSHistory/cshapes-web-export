Introduction

This Web-Application aims to facilitate the access to the cshapes dataset. For information on the dataset refer to the website and this article . Note that only sovereign states are included, and the dataset is mainly targeted at high resolution maps.

Usage

You can select either the Correlates of War (COW) or the Gledditsch and Ward (GW) reference dates. By changing the date you will instantly see the resulting geojson file displayed on the map. Use the download-Button to save the file to your computer.

Notes

If you are looking to extract multiple snapshots from the cshapes dataset via command line, you may consider the python-based cshapes-converter
